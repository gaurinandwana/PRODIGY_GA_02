# PRODIGY_GA_02
Generating images from text prompts using pre-trained models like Stable Diffusion and DALL-E-mini. Built with Hugging Face Diffusers and PyTorch. Supports both GPU and CPU. Simply provide a text prompt and get AI-generated images instantly. Great for learning text-to-image generation.


🌟Project Overview

This project demonstrates the use of pre-trained deep learning models to generate images from textual descriptions (text-to-image generation). By providing natural language prompts, the model produces visually rich and creative images, showcasing the capabilities of Generative AI.



🌟Objective

* To understand text-to-image generation
* To utilize pre-trained AI models for creative outputs
* To explore how prompts influence generated images
* To visualize and analyze generated results



🌟Tools & Technologies

* Python
* Pre-trained Deep Learning Model (Text-to-Image)
* Matplotlib (for visualization)
* Jupyter Notebook / Google Colab



🌟Methodology

1. Import Libraries

Essential libraries are imported for model handling and visualization.

2. Load Pre-trained Model

A pre-trained generative model is used to convert text prompts into images.

3. Define Prompts

Different descriptive prompts are provided as input to generate diverse outputs.

4. Generate Images

The model processes each prompt and generates corresponding images.

5. Display & Save Outputs

Generated images are visualized and saved for further use.
